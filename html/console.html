Console в DevTools

Виды сообщений в Console

  • Log Info — информационное сообщение.
  • Log Warning — предупреждающее сообщение.
  • Log Error — сообщение об ошибке.
  • Log Table — сообщение в виде таблицы.
  • Log Group — группы значений.
  • Log Custom — сообщение в произвольном формате.
  • Cause 404 — ошибка 404
  • Cause Error — произвольная ошибка.
